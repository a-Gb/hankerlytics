# Contributing

Thanks for helping improve HN Thread Atlas!

## Development

```bash
./start-dev.sh
```

Open `http://localhost:8080`.

## Guidelines

- Keep changes focused and documented.
- Use plain JavaScript and avoid unnecessary dependencies.
- Ensure large threads still perform well.

## Reporting issues

Include:

- A thread id or URL that reproduces the issue.
- Browser + OS.
- Console errors (if any).

## PR checklist

- [ ] Ran in the browser and verified core flows (load thread, pan/zoom, click).
- [ ] Updated README if behavior changed.
